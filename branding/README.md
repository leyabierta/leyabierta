## Colores

| Color | Hex | Uso |
|-------|-----|-----|
| Azul institucional | `#1a365d` | Accent, logo, enlaces |
| Texto principal | `#0b1120` | Titulos, cuerpo |
| Fondo | `#fafaf8` | Background |
| Verde | `#1a7a4e` | Vigente, texto nuevo |
| Rojo | `#b91c1c` | Derogado, texto eliminado |

## Tipografia

- **Source Serif 4** — titulos, encabezados, texto legal
- **Inter** — cuerpo, navegacion, UI
- **JetBrains Mono** — IDs, codigo, datos tecnicos
