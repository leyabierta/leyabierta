# Ley Abierta — Brand Assets

Todos los assets oficiales del proyecto. Usar siempre estos archivos como fuente.

## Logo

| Archivo | Uso | Preview |
|---------|-----|---------|
| `logo-full.png` | Logo completo con subtítulo (presentaciones, about pages) | ![](logo-full.png) |
| `logo-navbar.png` | Logo sin subtítulo (navbar, headers) | ![](logo-navbar.png) |
| `logo-icon.png` | Icono cuadrado (avatar de org, app icon) | ![](logo-icon.png) |

| Archivo | Uso |

## Colores

| Color | Hex | Uso |
|-------|-----|-----|
| Azul institucional | `#1a365d` | Accent, logo, enlaces |
| Texto principal | `#0b1120` | Títulos, cuerpo |
| Fondo | `#fafaf8` | Background |
| Verde | `#1a7a4e` | Vigente, texto nuevo |
| Rojo | `#b91c1c` | Derogado, texto eliminado |

## Tipografia

- **Source Serif 4** — títulos, encabezados, texto legal
- **Inter** — cuerpo, navegación, UI
- **JetBrains Mono** — IDs, codigo, datos técnicos
